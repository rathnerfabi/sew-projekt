# SEW Abschlussprojekt – Quiz

Quiz-Programm mit JavaFX für das SEW-Abschlussprojekt.

Man bekommt Fragen mit vier Antwortmöglichkeiten, wählt eine aus und sieht, ob sie richtig war. Am Ende gibt's Punkte und ein Ergebnis.

## Aktueller Stand

Das Spiel hat mehrere Fragen mit je 4 Antworten. Reihenfolge der Fragen und Antworten werden gemischt. Es gibt drei Ansichten (Menü, Quiz, Ergebnis), zwischen denen über den `ViewWechsler` umgeschaltet wird.
