# SEW Abschlussprojekt – Quiz

Quiz-Programm mit JavaFX für das SEW-Abschlussprojekt.

Man bekommt Fragen mit mehreren Antworten, wählt eine aus und sieht, ob sie richtig war. Am Ende gibt's Punkte und ein Ergebnis.

## Geplant

- mehrere Fragen mit Punktezählung
- Kategorien und Schwierigkeitsgrade
- Fragen aus Datei laden, Highscores speichern
- Auswertung (Trefferquote, beste Punktzahl)
